📝python的起源
1. python的起源：1989年，吉多范罗苏姆在圣诞节的假期决心开发一个新的解释器程序，python的雏形
2.1991年，第一个python解释器程序诞生了
3.python的名字：来自于吉多范罗苏姆喜爱的电视剧：Monty Python's FlyingCiircus

🌉 为什么选择python
1.优雅【简单，易学，开发效率高】
2.适用面非常广泛

🏷️ 什么是编程语言
1.概念：人类和计算机交流的一种专有领域语言
2.工作原理：将人类的想法，翻译成计算机可以识别的二进制，让计算机工作

📋 python的安装

💻 第一个python程序
1.print("Hello World") ----标点符号一定要英文的符号
&常见问题
^找不到命令提示符---快捷键：win+r---输入cmd
^python不是内部或外部命令---安装python的时候，没有勾选Add Python 3.xx.xx to PATH
^SyntaxError:invalid character---标点符号写成了中文符号

🌎 Python的解释器
1.计算机只认识什么？---0和1 二进制
2.Python解释器的作用---【将Python代码翻译成计算机能够识别的0和1 二进制】【解释器会将翻译的二进制提交给计算机去执行】
3.".py"---Python语言1的代码文件，里面记录Python的代码
4. python的解释器程序在哪里？---python的安装目录/python.exe
